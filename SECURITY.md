# Security Policy

## Supported versions

| Version | Supported          |
|---------|--------------------|
| latest  | :white_check_mark: |
| < latest | :x:               |

Only the latest published release receives security updates.

## Reporting a vulnerability

Report security vulnerabilities to **irving.frias+security@gmail.com**.

Do not use public GitHub issues for security reports.

You can expect:

- **Acknowledgement** within 48 hours
- **Status update** within 5 business days
- A fix will be released as soon as a patch is ready

Reports are kept confidential until a fix is published.
